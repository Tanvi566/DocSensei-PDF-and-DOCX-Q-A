# 📄 DocSensei

DocSensei is an AI-powered document question-answering application built using LangChain, ChromaDB, Google Gemini, and Streamlit.

## 🚀 Features

- Upload PDF and DOCX files
- Ask natural language questions
- Retrieval-Augmented Generation (RAG)
- Chroma Vector Database
- Google Gemini responses
- Source citations
- Two chunking strategies
- Adjustable retrieval (Top-K)
- "I don't know" responses for unsupported questions

## 🛠 Tech Stack

- Python
- Streamlit
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Google Gemini API

## Installation

```bash
git clone <repository-url>

cd DocSensei-PDF-and-DOCX-Q-A

pip install -r requirements.txt

streamlit run app.py
```

## Environment Variables

Create a `.env` file:

```text
GOOGLE_API_KEY=YOUR_API_KEY
```
